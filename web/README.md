# Web Khách Hàng - ViStay

> Tầng 3 trong cấu trúc 3 tầng: Web Layer (Customer-Facing)

## Mô tả

Thư mục này sẽ chứa giao diện web dành cho **khách hàng** của ViStay.

### Tính năng dự kiến:
- [ ] Trang chủ giới thiệu dịch vụ
- [ ] Đặt phòng online
- [ ] Tra cứu thông tin căn hộ
- [ ] Liên hệ & hỗ trợ
- [ ] Đánh giá dịch vụ

### Công nghệ dự kiến:
- HTML/CSS/JavaScript
- Kết nối API từ tầng Application Layer (`/server`)

> ⚠️ Sẽ bổ sung sau theo yêu cầu
